SecureMemo = Nishikaの**オンプレミス版**AI議事録アプリ(独自音声認識AI搭載)。出典: `2606_SecureMemo_サービス紹介資料_v80.pdf`(27p, 2026-06)。クラウド版は [[securememocloud-product]]。

## プロダクト概要
- 「文字起こし→要約まで完全オンプレ完結(完全オフライン対応)」が最大の売り。Windows専用アプリ。
- 主要機能: 高精度文字起こし / 完全自動の話者特定(声紋AI、同一マイクでも判別) / フィラー自動削除 / 多言語対応(100言語、文字起こし・翻訳) / 専門用語対応(辞書登録) / カスタム要約。
- 音声認識AI = 独自開発 **"shirushi"**(最先端OSS音声認識モデル + Nishika独自データでファインチューニング、句読点付与AI・話者特定AI追加)。精度: 高品質音声96.2% / 低〜中品質77.3%(他社55〜60%台に対し優位)。
- 2モデル搭載: **shirushi-p(高速・日本語のみ・CPUでも快適)** / **shirushi-w(多言語・GPU推奨)**。60分音声の文字起こし=高速GPU1〜2分/CPU10〜25分、多言語GPU4〜7分/CPU40〜75分。要約GPU4〜5分/CPU10〜20分。
- 利用方法2つ: ①リアルタイム認識(会議室に導入済PC設置) ②インポート認識(音声/動画ファイル投入)。インポートの方が高精度。

### クラサバ型のライセンス体系
- 基本パッケージ120万円/年 = **1サーバー+1アプリ+1AIライセンス**。
- 追加ライセンス: **アプリライセンス12万円/年**(AI機能=文字起こし/話者分離/要約 以外を同時利用) / **AIライセンス36万円/年**(AI機能を同時利用、=同時実行数) / **サーバーライセンス72万円/年**(別サーバーへのアプリ導入)。
- サーバーライセンスはボリュームディスカウント対応。複数契約もVD相談可。

### 共通条件
- 買い切りではなく**年額ライセンス**。アプリは約3ヶ月に一度アップデート(DL URL通知、DVD配布も可)。
- 特定国(日米)の開発モデルに限定したAI搭載。
- 保守範囲: Web/メールサポート(平日10-17時)、システム起因不具合の調査・復旧/代替案、製品バグフィックス。
