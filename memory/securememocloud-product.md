SecureMemoCloud = Nishikaの**クラウド版**AI議事録(会議録作成支援)サービス。出典: `2605_SecureMemoCloud_サービス資料_v45.pdf`(21p, 2026-05)。オンプレ版は [[securememo-onprem-product]]。プロダクトWikiは [[product-wiki-project]]。

## プロダクト概要
- 対面会議+全WEB会議(Zoom/Teams/Meet)に対応。「ほぼ完議事録」を生成。スマホアプリ+Webアプリ。
- 音声認識AI = 独自開発 **"shirushi"**(OpenAI Whisperを世界大会上位入賞のデータサイエンティストが改良)。精度: 高品質96.2% / 低〜中品質77.3%(オンプレと同値)。
- 主要機能: 文字起こし・校正 / 話者認識(声紋ベース、同一マイクでも可) / **【特許取得】単語登録の高精度変換**(音声AI×生成AI) / 議事録作成(複数テンプレ: ほぼ完議事録・カスタム要約等) / マイクで録音(全WEB会議対応) / 業界特化型音声認識(**23業界**) / 多言語音声認識(約100言語) / 多言語翻訳(β) / Web会議自動アップロード(現状zoomのみ)。

## セキュリティ(資料p15、大手企業向けの強み)
- 全データ**日本国内(東京)限定保管・国内法適用**。法人プランはAI学習に一切不使用。解約後30日以内に物理完全削除。
- AWS WAF導入(SQLi/XSS検知)、脆弱性診断「Securify」月1回以上、暗号化=保存AES-256/通信TLS1.3(電子政府推奨暗号準拠)。
- 日次バックアップ30世代、稼働率99.9%(目標)、重大インシデント1営業日以内通知。MFA/IP制限/監査ログ3年保持。
- **プライバシーマーク取得・ISO/IEC 27001(ISMS)取得・電子政府推奨暗号準拠**。詳細はセキュリティホワイトペーパーv1.7。
